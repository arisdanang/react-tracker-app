# React Tracker App

Demo: https://react-tracker-app.vercel.app/
This project was created with [Vite](https://vitejs.dev/).

It uses:
- **ReactJS,Hooks**
- **TailwindCSS**
- **Headless UI**  

## Installation

**Clone the repo:**
```bash
https://github.com/arisdanang/react-tracker-app.git
cd react-tracker-app
```

**Install the dependencies:**
```bash
npm install
```

**Running the app**
```bash
 npm run dev
```
